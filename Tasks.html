# Задание Краткое описание Ссылка на gitHub
1 Блочная вёрстка По примеру создать страницу с двумя блоками, шапкой и футером (добавить стили css в отдельном файле).
2 Вёрстка сайта по шаблону В соответствии с примером оформить контент страницы сайта (добавить стили css в отдельном файле).
3 JS. Задание 1 Создать страницу с использованием javasscript. Работа со ссылками, добаление текста на страницу, обращение к элементам, подсчет количества элементов, вывод их свойств. Добавить формы с кнопками (показать имя формы/ очистить форму и т.д.). Для каждой кнопки написать отдельную функцию. Создание динамической таблицы.
4 JS. Задание 2 Создать форму создания динамической таблицы. Пользователь вводит количество строк и столбцов. Добавить блок с функциями редактирования таблицы (изменения границ таблицы/ цвета ячеек/ удаление строк и т.д.). Для каждой кнопки написать отдельную функцию.
5 Практическая работа по jQuery Создать страницу с любым наполнением, изменить стили и активность элементов с использованием jQuery. Добавить несколько ссылок и заменить http на https, настроить кнопку отмены. Добавить таблицу с элементами, к которым применить разные типы анимации.
6 Практическая работа по Bootstrap Создать сайт из нескольких страниц. Доработать форму динамического создания таблицы c использованием фреймворка Bootstrap. Добавить страницу с подгрузкой карты Google Maps с определенной локацией и ее описанием, страницу с контактами автора, страницу с описанием выполненных заданий. Все оформить c использованием Bootstrap, адаптировать под разные размеры устройств.
